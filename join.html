Structure & function

A Borrelia burgdorferi lipoprotein

Investigate bacterial lipoprotein structure and function through protein production and high-resolution biophysics.

This project will involve

  • Overexpression of lipoproteins using an E. coli expression system
  • Structural and functional studies using X-ray crystallography and cryo-EM

02

Biomaterials

A novel lipoprotein-based material

Develop and characterize a new protein-based biomaterial using complementary structural and biophysical approaches.

This project will involve

  • Overexpression of lipoproteins using an E. coli expression system
  • Characterization using cryo-EM, negative-stain TEM, nanoDSF, and DLS
